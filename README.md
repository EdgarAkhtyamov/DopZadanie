# DopZadanie

Описание работы программы:
1) создаем двумерный массив
2) инициализируем и выделяем под него память с помощью функции memory
3) распределяем первое поколение живых клеток по массиву, используя функцию first_gen, которая случайно заполняет каждую клетку массива числами 0 или 1, где 0 - мертвая клетка, 1 - живая клетка
4) программа будет выполняться до тех пор, пока булевая переменная flag не примет значение false, что будет говорить о конце игры и невыполнением условий
5) в цикле while преобразуем наш массив с помощью функции до 2... поколения, используя функцию generation, которая проходит по всем клеткам, считает количество соседей с помощью функции count_lifes, выполняет определенную последовательность действий исходя из количества живых соседей
6) далее выполняется булевая функция game_over, которая проверяет, удовлетворяет ли какому-нибудь из условий наша вселенная. Если удовлетворяет, то переменная flag принимает значение 0 и программа завершает свое выполнение
7) Если же ни одно из условий в game_over не выполяется, то наша вселенная выводится на экран с помощью функции demo
